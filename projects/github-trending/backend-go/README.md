The Github Trending Repositories REST API with Golang.
This demonstration describes how to:

- Use Echo package for REST API creation
- Migrate MySQL database from scheme files
- Use repository design pattern
- Use struct validator in Go
- Use JWT in Go
- Use Redis for caching in Go
- Build a crawler with colly package
- Build a job queue with channels
- Write logs to files
- Use docker compose to build a Go service and Redis service
